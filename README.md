# Phaser Vite TypeScript Electron Template

## Быстрый старт

Для тестирования приложения в браузере

```
yarn web
```

Для запуска в оконном режиме

```
yarn dev
```

Билд

```
yarn install
yarn build
yarn make
```

Готовое приложение будет лежать в папке

```
/out/template-vite-ts-win32-x64/template-vite-ts.exe
```
